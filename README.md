# mvw.js

A minimalist MVW framework for non-upgradeable dApps.

## Overview

mvw.js is based on few highly consolidated browser technologies such as raw DOM, `<template>`
elements, and template strings. It has zero dependencies. The compressed JavaScript file takes less
than 1 KB.

## Motivations

Libernet's mission is **full-stack decentralization**. Some dApps are upgradeable but some are not,
therefore some developers need to develop dApps (including both the backend _and_ the frontend) that
can never be changed and keep working for the foreseeable future. Using complex frameworks such as
React, Next.js, Vue.js, etc. which are available for general web development is unadvisable for
non-upgradeable dApps, as their complex structure may require upgrades over time due to security
vulnerabilities, the use of unstable browser features, or other reasons.

mvw.js is a lightweight and very stable library based on a few highly consolidated browser
technologies: raw DOM, ES6, and `<template>` elements. mvw.js is designed with avoiding planned
obsolescence as a first principle.

mvw.js's browser coverage exceeds 96% based on caniuse.com data, and can only grow over time.
